# Education And Course Platforms

## Core Workflows
- Audit course, lesson, faculty, and resource pages for crawl depth and discoverability.
- Detect orphan course pages and broken links between catalog, curriculum, and enrollment paths.
- Validate canonicals and redirects across archived terms, retired courses, and new cohorts.
- Monitor duplicate metadata patterns generated by program templates.
- Analyze internal link equity between degree hubs, category pages, and long-tail landing pages.

## Automation Ideas
- Build a scheduled report of orphan or redirected course pages.
- Diff crawls each term to catch accidental noindex or slug changes.
- Segment audits by school, faculty, degree type, or locale using path rules.
- Join crawl data to enrollment priorities to fix broken paths on strategic programs first.

## Why This Library Fits
- Education sites combine template sprawl with periodic content churn.
- Crawl SQL helps analyze large catalogs without repeated exports.
- Diffing is practical for term-based launches and sunsetting old content.
