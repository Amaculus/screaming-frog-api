# Localization And International SEO

This file focuses on locale, market, and hreflang operations.

## Core workflows
- Missing return-link detection.
- Noindex hreflang target detection.
- Non-canonical hreflang target detection.
- Missing self-reference detection.
- Missing x-default detection.
- Locale parity checks for titles, canonicals, and indexability.

## Team workflows
- Route issues to market owners.
- Summarize issue counts by country folder or subdomain.
- Diff locale health between releases.
- Build launch QA packs for new markets.

## Why this is valuable
- International issues often sit in edge-case reports that are slow to review manually.
- Code makes it practical to rerun the same market checks every crawl.
- Locale owners get focused outputs instead of generic crawl exports.
